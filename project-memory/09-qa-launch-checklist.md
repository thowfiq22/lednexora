# QA And Launch Checklist

Last updated: 03-05-2026

## Content QA

- All accepted pages exist.
- Page names match accepted sitemap.
- No old out-of-scope page appears in navigation.
- Zoho content reduced to accepted 3 pages.
- Employee onboarding page exists and works.
- Content is short, clear, and professional.
- No placeholder text remains.
- No fake testimonials/certifications/logos.

## Visual QA

- Blue/white theme consistent.
- Gradient accents are controlled and professional.
- Source Sans 3 or approved Rimini-like free font loads correctly.
- Design resembles enterprise consulting quality.
- Header and footer are consistent.
- Buttons and CTAs are clear.
- Mobile layout is clean.
- Text does not overflow cards/buttons.

## SEO QA

- Unique page titles.
- Meta descriptions present.
- H1 exists once per page.
- Open Graph metadata present.
- Sitemap exists.
- Robots.txt exists.
- Important old URLs redirected if needed.

## Functional QA

- Contact form validates and submits.
- Contact email notification works.
- Spam protection works.
- WhatsApp quick-contact works.
- Onboarding form validates and submits.
- Onboarding data saves if database enabled.
- Onboarding notification works.
- Admin login works.
- Admin content publish/unpublish works.
- Insights listing and detail pages work.

## Browser/device QA

Test at minimum:

- Chrome desktop
- Edge desktop
- Android mobile viewport
- iPhone mobile viewport if possible
- Tablet width

## Deployment QA

- Backup current site.
- Upload final files.
- Database imported.
- Config updated.
- Forms tested on production hosting.
- Admin tested on production hosting.
- SSL active.
- DNS/domain behavior verified.
- Final smoke test completed.

## Handover QA

- Admin credentials created.
- Basic admin usage notes prepared.
- Renewal responsibility clarified.
- Support period noted.
- Change log updated.

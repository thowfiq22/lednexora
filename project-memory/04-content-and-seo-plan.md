# Content And SEO Plan

Last updated: 03-05-2026

## Content direction

Client confirmed content should be revised and refined to be:

- shorter
- clearer
- more professional
- more precise
- better structured for consulting credibility

Do not paste the old content unchanged if it is long, weak, or repetitive. Reuse the meaning, but refine the presentation.

## Tone

Use international B2B consulting English:

- clear
- direct
- confident
- not exaggerated
- not overly technical unless the page needs it
- UK-friendly for Zoho where relevant, but not forced UK-only

## SEO approach

Use basic SEO for every page:

- unique page title
- unique meta description
- one H1 per page
- clear H2/H3 hierarchy
- meaningful URL slug
- Open Graph title/description/image
- internal links between related pages
- sitemap and robots.txt

## Zoho SEO note

Zoho section should support UK-based search intent where natural.

Possible phrases to use carefully:

- Zoho bookkeeping services
- Zoho BPO support
- Zoho consulting partner
- Zoho implementation support
- Zoho support for UK businesses
- outsourced bookkeeping support
- Zoho business solutions

Do not overstuff keywords. The page must read like a professional consulting website.

## SAP SEO note

SAP pages should cover:

- SAP consulting
- SAP advisory
- SAP support
- SAP implementation guidance
- SAP optimization
- SAP business transformation
- industry-specific SAP solutions


## Source screenshot usage

Before writing or rewriting any page, check `12-source-asset-inventory.md` and the relevant screenshot folder.

Use old Lednexora screenshots for:

- business meaning
- existing service categories
- old content clues
- navigation/content relationships

Do not use old screenshots for:

- final visual style
- final section spacing
- final typography
- final image quality

Use Rimini reference screenshot for structure and presentation quality, not for copying text or assets.

## Content workflow per page

For each page:

1. Review old screenshot/content source.
2. Extract the core message and useful service points.
3. Rewrite into short professional consulting copy.
4. Align section structure with `03-page-by-page-section-plan.md`.
5. Add SEO title/meta/H1 draft.
6. Mark missing proof/content/assets as TODO instead of inventing.
## Content needed from client

- approved company description
- service descriptions
- SAP service details
- Zoho service details
- client logos
- testimonials
- certifications / authorization proof
- contact details
- social links
- onboarding form fields
- images/media assets
- privacy/terms/accessibility legal copy or approval

## Content rewrite rules

Do:

- convert long paragraphs into short sections
- use proof and outcomes where possible
- make CTA copy clear
- keep services easy to scan
- use real business terms, not generic filler

Do not:

- invent certifications
- invent client logos/testimonials
- make unsupported claims
- copy text from reference websites
- use heavy jargon without context
